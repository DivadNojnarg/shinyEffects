> Fancy CSS effects for shiny

Customize your shiny apps with these nice CSS effects

## Installation

```r
# from CRAN
install.packages("shinyEffects")
# Latest version
devtools::install_github("RinteRface/shinyEffects")
```

## Demo

```r
library(shinyEffects)
shinyEffectsGallery()
```
